🚴‍♂️ FitCycle — 飲食 × 運動 × 目標追蹤系統

FitCycle 是一個以 React + TypeScript + Redux Toolkit + Google Apps Script（GAS） 打造的個人飲食與訓練紀錄系統。
系統會依據使用者的 身高、體重、年齡 自動計算 TDEE 與三種日制（低碳 / 中碳 / 高碳）的建議攝取，並提供每日「攝取是否達標」比對功能。
